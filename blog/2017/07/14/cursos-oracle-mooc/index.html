Cursos Oracle MOOC

14/7/2017 :: cursos, oracle, mooc, java

A Oracle está oferecendo dois cursos sobre Microservices e Cloud Services…

Em 17 de julho de 2017 começa o curso Java Microservices Using Oracle Cloud - Session 2 - matrícula AQUI, o curso possui 3 semanas e aborda os seguintes conteúdos:

  • Introdução para REST e Microservices;
  • Aplicação RESTful com Spring Boot; e
  • Oracle Developer Cloud Services.

Já em 07 de agosto de 2017 começa o curso Introduction to Java Cloud Services - matrícula AQUI, o curso possui 3 semanas e aborda um material sobre Oracle Cloud.
