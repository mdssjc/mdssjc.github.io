Instalação do Emacs
Clone o repositório do Emacs no Github:
git clone https://github.com/emacs-mirror/emacs
Compile e instale o Emacs:
./autogen.sh all
./configure
make
sudo make install
Para atualizar:
git pull
 E repita a etapa anterior.
