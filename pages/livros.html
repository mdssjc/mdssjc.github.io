Livros

Online

  • The Nature of Code: livro de Processing com estratégias e técnicas de programação sobre simulação computacional.
  • How to Design Programs: livro de Racket sobre metodologia de desenvolvimento de software.

Arquitetura de Software

Introdução

  • Head First Object-Oriented Analysis and Design: livro de Brett McLaughlin, Gary Pollice, David West; apresenta os conceitos e técnicas de Análise e Projeto Orientado a Objetos com exemplos práticos e slides e exercícios suplementares.
  • Head First Software Development: livro de Dan Pilone, Russ Miles; apresenta um processo de desenvolvimento com metodologia ágil, baseado no Extreme Programming, com exemplos práticos.
  • Head First Design Patterns: livro de Eric Freeman, Elisabeth Robson, Bert Bates, Kathy Sierra; apresenta os padrões de projetos (Design Patterns) do GoF (Gang of Four) e a construção do MVC (Model-Controller-View) com exemplos práticos e comparação com implementações internas do JDK (Java Development Kit).

Referência

  • Design Patterns: Elements of Reusable Object-Oriented Software: livro sobre padrões de projeto (Design Patterns) do GoF (Gang of Four).

Carreira

  • The Pragmatic Programmer from Journeyman to Master: livro de David Thomas, Andrew Hunt; apresenta um grande compilado de ensinamentos para fortalecer o conhecimento e experiência do leitor, altamente recomendável para desenvolvedores de diferentes níveis de conhecimento.

Java

  • Pragmatic Unit Testing in Java 8 with JUnit: livro de Jeff Langr, Andy Hunt, Dave Thomas; apresenta os conceitos sobre testes unitários com o Java 8, abordando as principais técnicas e ferramentas em um caso de uso.
  • Functional Programming in Java Harnessing the Power Of Java 8 Lambda Expressions: livro de Venkat Subramaniam; apresenta uma abordagem dos recursos de programação funcional apresentados no Java 8 para os problemas diários encontrados pelos desenvolvedores.
