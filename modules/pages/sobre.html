O que é?

A Página apresenta diversos materias de estudo sobre o tema Arquitetura de Software e Testes, tendo como base o processo de desenvolvimento do aplicativo Citações.

A compilação do estudo será através do Livro e os objetos de estudo através do Repositório (ambos referenciados pela barra de navegação - no superior da página).

Tecnologias

O site utiliza as tecnologias básica da Web, tais como HTML (versão 4 por enquanto), CSS e JavaScript, assim o mesmo serve como material de estudo - também presente no repositório.

Existe a possibilidade de utilização de banco de dados por chave (como Firebase) - assim permitindo uma interação com os leitores, esse com maior uso pelo JavaScript.

A codificação é realizada com o Emacs (Spacemacs), principalmente com uso do Emmet.

A monitoração é realizada pelo Google Analytics - os dados serão sumarizados para publicação e o aprendizado, com o crescimento do site.

Resumo

  • HTML: 4.01 (breve HTML 5)
  • CSS: 3
  • JavaScript: scripts do Google
